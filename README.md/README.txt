# Análise: O impacto dos videogames no desempenho escolar 🎮📊

Este projeto explora um conjunto de dados do Kaggle para entender a relação entre tempo de jogo e notas dos alunos.

## 📂 Estrutura do projeto:
- **notebooks/** → Contém as análises feitas no Google Colab
- **datasets/** → Contém os arquivos CSV usados na análise
- **images/** → Contém gráficos e prints gerados
- **scripts/** → Contém códigos auxiliares em Python

## 🚀 Tecnologias usadas:
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- GitHub

## 📊 Principais descobertas:
✅ Correlação fraca negativa entre tempo de jogo e notas (-0.1576)  
✅ Alunos que jogam menos tendem a ter notas mais altas  
✅ Escolaridade dos pais impacta o desempenho acadêmico
